Formats supportés : .csv (séparateur virgule ou point-virgule)

Format attendu

Le fichier CSV doit contenir les colonnes suivantes (ordre libre) :

  • nom (obligatoire) – Nom du produit
  • sku (obligatoire) – Référence unique
  • categorie (facultatif) – Nom de la catégorie (sera créée si inexistante)
  • prix_unitaire – Montant en FCFA (ex: 12500)
  • seuil_alerte – Quantité minimale (par défaut 0)
  • unite_mesure – Unité (unite, kg, litre, boite, etc.)
  • code_barres – Code EAN optionnel
  • description – Texte libre
  • fournisseur_principal – Nom du fournisseur (ignoré si non trouvé)

Conseils

  • Vérifiez que l’encodage est UTF-8 pour éviter les caractères mal affichés.
  • Les champs obligatoires sont nom et sku.
  • Le SKU doit être unique par entreprise → en cas de doublon, l’option choisie détermine le comportement.
  • Les produits créés par import ne génèrent pas de stock initial (il faudra ensuite créer un mouvement manuellement).
